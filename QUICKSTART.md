---

## 1. Получение токена бота (2 минуты)

### Шаг 1: Создайте бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Введите имя бота (например: "My Chat Monitor")
4. Введите username (должен заканчиваться на "bot", например: "my_chat_monitor_bot")
5. **Сохраните токен** (выглядит так: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Получите ваш Telegram ID

1. Откройте бота **@userinfobot**
2. Нажмите `/start`
3. **Скопируйте ваш User ID** (только цифры)

## 8. Часто задаваемые вопросы

### Бот не отвечает на команды

**Причина:** Бот не запущен или неверный токен

**Решение:**
```bash
# Проверьте, что бот запущен
ps aux | grep bot.py

# Проверьте токен в bot.py
grep "BOT_TOKEN" bot.py

# Перезапустите
python bot.py
```

### Бот не отслеживает удаленные сообщения

**Причина:** Не подключен через Telegram Business или нет Premium

**Требования:**
- ✅ Telegram Premium
- ✅ Подключение через Настройки → Чат-боты
- ✅ Активная подписка бота

### Медиафайлы не сохраняются

**Решение:**
```bash
# Создайте директории
mkdir -p media database exports

# Проверьте права
chmod 755 media database exports

# Проверьте место на диске
df -h
```

### Ошибка "Unknown connection_id"

**Причина:** Подключение не сохранено в БД

**Решение:**
1. Отключите бота в Настройках
2. Подключите заново
3. Проверьте уведомление о подключении

### Платежи не работают

**Проверка:**
- Правильно ли указан BOT_TOKEN?
- Включены ли платежи в @BotFather?
- Есть ли у пользователя Stars?
