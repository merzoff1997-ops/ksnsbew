### ⚠️ ВАЖНЫЕ ПРИМЕЧАНИЯ

#### Ограничения Telegram Business API

**НЕ поддерживается:**
- ❌ Секретные чаты (Secret Chats) - ограничение Telegram API
- ❌ Групповые чаты (Group Chats) - Business API только для личных
- ❌ Каналы (Channels) - Business API не работает с каналами
- ❌ Чаты без явного подключения

**Поддерживается:**
- ✅ Личные чаты (Private Chats) через Business API
- ✅ Медиа с таймерами самоуничтожения
- ✅ View-once медиа (одноразовый просмотр)
- ✅ Кружки (video notes)
- ✅ Все типы медиа

#### Требования

- ✅ Telegram Premium (обязательно)
- ✅ Подключение через "Настройки → Чат-боты"
- ✅ Активная подписка бота (после пробного)

#### Миграция с v6.0

**Автоматическая миграция:**
- База данных обновится автоматически
- Добавятся новые таблицы и поля
- Старые данные сохранятся

**Что НЕ требуется:**
- Переустановка бота
- Повторное подключение чатов
- Ручная миграция данных

**Рекомендации:**
- Сделайте backup БД перед обновлением
- Проверьте логи после первого запуска
- Протестируйте основные функции
