## 2. ОПИСАНИЕ СЕРВИСА

2.1. **Основной функционал:**
- Сохранение всех сообщений из подключенных чатов
- Уведомления об удаленных сообщениях
- Отслеживание изменений сообщений
- Сохранение медиафайлов, включая с таймерами самоуничтожения
- Экспорт данных

2.2. **Принцип работы:**
- Бот подключается через функцию "Чат-боты" в настройках Telegram
- Пользователь сам выбирает, какие чаты подключить
- Бот работает только с явно подключенными чатами
- Нет доступа к другим чатам пользователя

2.3. **Ограничения:**
- Бот сохраняет только новые сообщения после подключения
- Нет доступа к сообщениям до момента подключения
- Требуется активная подписка для полного функционала
- Требуется Telegram Premium для использования функции "Чат-боты"
